This gallery is a collection of sketches made while learning p5.js.

For those unaware, p5.js is a Javascript library that allows you to code up very fascinating behavior like Brownian motion, particles, flow fields, very interesting stuff. Here, however, I used the simplest code they encourage you to start with. Infact, it should be understandable to nearly anyone. 

```
function setup() {
  createCanvas(400, 400);
}
function draw() {
  //when mouse button is pressed, circles turn black
  if (mouseIsPressed === true) {
    fill(0);
  } else {
    fill(255);
  }

  //white circles drawn at mouse position
  circle(mouseX, mouseY, 100);
}
```

Don't worry, though, it's not necessary to understand it to appreciate the art below. 

They were created with the intention of making modern art, because I have also thought, "Hey, I could have made that!". However, someone on Tumblr said, "Go ahead, make whatever you want to make, splatter paint around if you want." So I made these by moving a circle around that leaves a trail, but if you press it, it leaves black circles.

Within some time, I felt like I was consciously moving towards certain shapes and ideas, and surprisingly I even discovered my friends had preferences for what they liked over another.

![The drifter spirals.](/content/weird/p5/drifter.jpeg)

This one is my personal favorite. They say each artwork is different for each perceiver, and I perceive this sort of tearing away from the tide, forging a lone path and I think this seems to capture that.
