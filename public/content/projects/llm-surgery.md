Developed LLM Surgery, aimed at creating and utilizing LLM agents capable of performing various mechanistic interventions on other language models.

Conducted experiments ranging from an agent unsteering a mechanistically steered model back to a neutral state to an agent performing mechanistic edits that produced a custom LLM according to user requirements.

Utilized Goodfire's API together with its predefined functions to create the actions available to the agents.
