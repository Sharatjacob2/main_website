Developed The Full Explanation, an interface built using HTML, CSS, JavaScript and Flask that serves as a wrapper around API calls to Mistral-7B-Instruct hosted on Hugging Face.

The interface provides explanations for user-specified topics and recursively generates further explanations for any complicated terms appearing within previous explanations.
